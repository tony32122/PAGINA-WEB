Front End

-Es visible

-Interactua el usuario.

-Es todo lo que se ve en una pantalla de un sitio web.

-Se encarga de la experiencia del usuario.

-Es sencillo de usar y funcional.

Back End

-No es visible

-Logica de Negocios

-Un servidor y una base de datos.

-Se toman los datos, se procesa la información y se envía al usuario.

-Es seguro y rendidor

API

(interfaz de Programacion de Aplicaciones)

-Es cómo un módulo de un software que se comunica o interactúa con otro para cumplir una o muchas funciones.

-Permite la comunicación entre dos aplicaciones de software a través de un conjunto de reglas.

-Facilita el trabajo a los desarrolladores.

-Utiliza las funciones de determinado servicio para ofrecer ventajas a sus usuarios.

Base de Datos

-Entidad en la cual se almacenan datos de manera estructurada y con la mínima repetición posible.

-Logica de Negocios

-Un servidor y una base de datos.

-Se toman los datos, se procesa la información y se envía al usuario.

SQL

-Permite combinar de forma eficiente diferentes tablas para extraer información relacionada.

-Crear y modificar el diseño de objetos de base de datos, como tablas.

-Tiene capacidad de hacer cálculos avanzados y álgebra.

NOSQL

-Permite distribuir grandes cantidades de información.

-Permiten adaptarse a necesidades de proyectos mucho más fácilmente que los modelos de Entidad Relación.

-Suelen ser bases de datos mucho más abiertas y flexibles.
